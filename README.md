### Instalasi

dapatkan [[Gemini API Key](https://aistudio.google.com/app/apikey)] untuk .env

1. Clone the repo
   ```sh
   git clone https://github.com/marsabilaaa/tira-chatbot.git
   ```
2. Install Dependencies
   ```sh
   npm install
   ```
3. Run dev server
   ```sh
   npm run dev
   ```
